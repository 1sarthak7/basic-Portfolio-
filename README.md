#  Basic HTML Portfolio

A clean, minimal, and responsive **personal portfolio website** built using **pure HTML and CSS**. This project is designed as a beginner-friendly portfolio that highlights personal information, skills, projects, and contact details in a professional and visually appealing layout.

---

##  Project Overview

This portfolio website serves as a simple personal branding page. It focuses on:

* Clean UI design
* Readable layout
* Responsive behavior for mobile devices
* Easy customization for beginners

The design uses a **dark theme**, modern typography, and card-based sections to keep the content structured and elegant.

---

##  Technologies Used

* **HTML5** – Structure and content
* **CSS3** – Styling, layout, responsiveness
* **CSS Variables** – Easy theme customization
* **Flexbox** – Skills layout and alignment


---

##  Project Structure

```
├── index.html
└── README.md
```

* `index.html` → Main portfolio webpage
* `README.md` → Project documentation

---

##  Sections Included

* **Header** – Name and professional title
* **About** – Short personal introduction
* **Technical Skills** – Skill tags displayed in a grid
* **Projects** – Highlighted project cards with descriptions
* **Contact** – Email contact link
* **Footer** – Copyright information

---

##  Responsive Design

The layout adapts automatically for:

* Desktop screens
* Tablets
* Mobile devices

---

##  How to Run the Project

1. Download or clone the repository

   ```bash
   git clone https://github.com/1sarthak7/basic-html-portfolio.git
   ```
2. Open `index.html` in any modern web browser


---

##  Customization Guide

You can easily customize:

* **Name & title** in the header section
* **Skills** by editing skill tags
* **Projects** by adding or modifying project cards
* **Colors & theme** using CSS variables in `:root`
